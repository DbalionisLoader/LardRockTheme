Welcome to LardRock theme - A custom Wordpress theme with a slick name.

FILE structure:
404.php - Error
archive.php - Listing page
search.php - search result page
comments.php - blog comments section
header.php - navbar
footer.php - page footer
index.php - main fallback page
page.php - static page template
single.php - a single blog post page
style.css - global styling
screenshot.png - theme image
function.php - custom php function and overrides
front-page.php - homepage/landing page
